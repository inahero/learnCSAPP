### IEEE 浮点表示

#### 计算公式V：实际的值，s:符号指数，M:底数，E:指数  
```
v = (-1)^s * M * w^E
```
#### 单精度二进制组成  
```
s  exp  frac  
1  8    23
```
#### 双精度二进制组成  
```
s  exp  frac  
1  11   52
```

#### 标准的情况
```
特征：exp 不全是0也不全是1
s = s
M = 1 + frac(小数表示)
E = exp(无符号) - (2^(exp位数少1) - 1)
    e.g:设exp:=0110  E = 0110 - 0111 = -1
```
#### 非标准情况
```
特征：exp 全是0
s = s
M = frac(小数表示)
E = 1 - (2^(exp位数少1) - 1)
```
#### 特殊值
```
特征：exp 全是1
正无穷: frac全是0, s=0
负无穷: frac全是0, s=1
NaN: frac不全是0
```
